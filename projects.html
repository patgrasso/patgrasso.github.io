BoxChat

Boxchat is a NodeJS and socket.io powered chat client built for teams.

(Similar to Slack)

  • Node.js and Express run the server
  • Socket.io allows real time client-server communication
  • Passport handles authentication
  • MongoDB used to store chat logs, user data, etc.
